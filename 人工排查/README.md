# 妊娠期体重人工排查工具

该工具用于在本地逐样本查看体重轨迹，并对选中点执行三种人工修正：
- `×2`
- `÷2`
- `删除该时间点记录`

并支持“疑似筛查模式”：
- 一键切换“仅显示疑似异常点”
- 可调“跳变阈值(kg)”（默认 `8`）
- 全量模式下仍会高亮疑似点，便于快速定位

## 安全机制

- 原始文件默认只读，不会被直接覆盖。
- 每次操作都会追加到 `data/edit_log.csv`（可回放）。
- 最终结果通过“导出修正CSV”生成新文件。

## 默认输入文件

程序默认读取：

`..\清洗管线重构_三步走\04_首尾异常与分娩校准版.csv`

也可以在界面里点击“选择CSV”切换输入文件。

- Python: `D:\miniconda3\envs\py3.10\python.exe`
- 依赖见 `requirements.txt`

## 启动

在当前目录执行：

```powershell
D:\miniconda3\envs\py3.10\python.exe .\manual_weight_review.py
```

或双击 `run_review.bat`。

## 操作流程

1. 在“样本ID”输入框输入 `项目流水号`，按回车或点“加载样本”。
2. 在左侧轨迹图点击点位，或在右侧表格点击对应行。
3. 点击 `×2` / `÷2` / `删除该记录`。
4. 勾选“仅显示疑似异常点”可进入快速筛查模式；可改“跳变阈值(kg)”并点“应用阈值”。
5. 必要时点击“撤销上一步”。
6. 点击“导出修正CSV”生成人工校正后的新表。

## 输出目录

- 导出文件：`exports/`
- 操作日志：`data/edit_log_<输入文件名>.csv`

## 备注

- 删除操作为逻辑删除（在工具内标记删除，导出时真正剔除）。
- `×2`/`÷2` 会同步更新 `weight`，若存在 `height` 与 `BMI` 列，会自动重算 `BMI`。
- 疑似点规则（当前版本）：
  - 体重超出 `[30, 140]kg`
  - 或与相邻记录体重差值 `>= 跳变阈值(kg)`
